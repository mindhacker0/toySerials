### 一. node 是什么？

Node.js 是⼀个 JS 的服务端运⾏环境，是在 JS 语⾔规范的基础上，封装了⼀些服务端的runtime，让我们能够简单实现⾮常多的业务功能。
Node.js 在2009年（第一版npm被创建）诞生之初是为了实现高性能的 web 服务器，再后来 Node.js 慢慢演化为了一门服务端“语言”。
